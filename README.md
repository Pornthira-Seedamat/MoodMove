## ความสามารถของระบบ (Features)

**Dashboard**
*ระบบคัดกรองอารมณ์ (Mood Selection)*
 หน้าเลือกอารมณ์ที่ตอบโจทย์ความรู้สึก ณ ขณะนั้น 
 - โกรธ
 - เศร้า
 - ปกติ
 - ดีจัง
 - สุดยอด

*ใช้ระบบ Animation (Framer Motion)*
 - เพื่อสร้างประสบการณ์ที่ลื่นไหล

*โปรแกรมออกกำลังกายส่วนบุคคล (Personalized Exercises)*
*ระบบแนะนำท่าออกกำลังกายที่เหมาะสมกับอารมณ์*
 - เช่น อารมณ์โกรธจะแนะนำการระบายพลังงานแรงๆ)

*แบ่งขั้นตอนการออกกำลังกายเป็นลำดับ (Step-by-Step)*

**history**

*ระบบบันทึกและแสดงประวัติ (Mood History & Statistics)*

*หน้าประวัติ (History)*
 - แสดงรายการบันทึกอารมณ์ย้อนหลัง


**status**

*กราฟสถิติการออกกำลังกายรายเดือน (Mood Graph)*

 - แสดงความเปลี่ยนแปลงของอารมณ์ในรูปแบบเส้นกราฟ

*รองรับการแก้ไขและลบข้อมูล (Edit & Delete Mode)*

*ระบบฐานข้อมูลผู้ใช้งาน (User Management)*

*จัดเก็บข้อมูลประวัติรายบุคคลลงใน Database (Prisma + PostgreSQL)*

*มีการจำค่าผู้ใช้งานผ่านระบบ API และ LocalStorage*

## เทคโนโลยีที่ใช้ (Tech Stack)
**Frontend**
- Next.js (App Router): เฟรมเวิร์กหลักสำหรับการจัดการโครงสร้างเว็บไซต์
- React: ใช้จัดการ UI และ State ภายในแอปพลิเคชัน
- Tailwind CSS: ใช้สำหรับการออกแบบสไตล์และ Layout ที่สวยงาม
- Framer Motion: ใช้สำหรับงาน Animation และ Transition ต่างๆ
- Lucide React: ชุดไอคอนสำหรับตกแต่งส่วนติดต่อผู้ใช้

**Backend**
- Next.js API Routes: จัดการระบบ Server-side logic และการเชื่อมต่อ API
- Prisma ORM: ตัวกลางในการสื่อสารและจัดการข้อมูลในฐานข้อมูล
- PostgreSQL: ระบบฐานข้อมูลเชิงสัมพันธ์ที่ใช้เก็บข้อมูลจริง (เช่น ผ่าน Supabase หรือ Neon)

**Deployment**
- Vercel: ใช้สำหรับการ Deploy โปรเจกต์ขึ้นสู่ระบบออนไลน์แบบ Production
